Shakespeare was an influential writer in the era of english monarchies. Born in 1564 and death in 1616; Shakeaspeare wrote many plays throughout the years that changed high school english classes for centuries


The main objective of the application is that when a seed is given to the application the code builds a script similar to old Shakespeare plays. The final set of data consists of 60 epochs or datasets with which to derive a partial play script will characters and dialogue. This app demonstrates procedural text generation using machine a learning techique call LSTM (Long Short Term Memory). It is trained by reading all of Shakespear's works and learns to emulate the writing style of the Bard.

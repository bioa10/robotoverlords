In this project we illustrate the core functionality in machine learning using python. In just a single script the dataset is loaded, the model is trained, then validated, and a few cases are caught to prove its effectiveness. The dataset is quite simple. Comprised of image data and their corresponding numeric value, you can view examples of it here. Because the train script handles all the code associated with managing a model, it can be quite can come with quite a wait time. One can view the epochs inside the console from which this web app is running to track the progess of an individual train.
